Request

Endpoint: https://capc.us
Method: POST
Content-Type: application/json
Body:
{
    "id": string,
    "type": "capcus.create",
    "url": string,
    "owner": string
}
id: request id (ex: "1945")
url: source url (ex: "https://www.krybrig.org/jejak.html?token=ABCxyz")
owner: anonymous or google account token (see: API Token)

Example:
{
    "id": "1945",
    "type": "capcus.create",
    "url": "https://www.krybrig.org/jejak.html?token=ABCxyz",
    "owner": "anonymous"
}

Response

Body:
{
    "source-url": string,
    "target-url": string,
    "create-time": string,
    "expire-time": string
}
source-url: source url (ex: "https://www.krybrig.org/jejak.html?token=ABCxyz")
target-url: your short url (ex: "https://capc.us/ABCDwxyz")
create-time: creation time (ex: "2018-11-08 05:03:57")

expire-time: expiration time (ex: "2018-11-22 05:03:57")

Example:
{
    "source-url": "https://www.krybrig.org/jejak.html?token=ABCxyz",
    "target-url": "https://capc.us/ABCDwxyz",
    "create-time": "2018-11-08 05:03:57",
    "expire-time": "2018-11-22 05:03:57"
}
